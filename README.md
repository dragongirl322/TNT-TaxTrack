# TNT Tax Tracker

A browser-based finance tracker for TNT Music. The app is designed to run in any modern browser and can be deployed on Railway with a simple Node.js server.

## Run locally

```bash
npm install
npm start
```

Then open http://localhost:3000.

## Deploy on Railway

1. Connect this repository to Railway.
2. Set the build command to `npm install`.
3. Set the start command to `npm start`.
4. Railway will expose the app on a public URL automatically.
